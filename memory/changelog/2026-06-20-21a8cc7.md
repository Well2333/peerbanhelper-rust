# feat: completed_size 填充 + /blocklist/ip 导出

## 做了什么
- **completed_size**：qB `/torrents/info` 返回 `completed`(已完成字节数),`QbTorrent` 加 `completed: Option<i64>`,种子 `completed_size = completed.unwrap_or(-1)`。**启用 PCB excessive Case2**(未完成任务上传 > 已完成量×倍率)。
- **`GET /blocklist/ip`**：公开纯文本端点(无需鉴权),每行一条当前封禁的 IP/CIDR,供下载器/外部消费。

## 验证
- clippy/fmt 干净;`cargo test --workspace` **86 单测全绿**(新增 qB 种子解析 completed 测试)。

## 注意
- completed 缺失(老 qB)时 -1,PCB Case2 跳过(Case1 仍有效),与原行为兼容。
- /blocklist/ip 公开暴露当前封禁 IP(低敏感),与上游 `/blocklist/*` 一致。
