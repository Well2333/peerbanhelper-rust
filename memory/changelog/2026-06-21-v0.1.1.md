# v0.1.1：WebUI 重构与一批面向用户的能力

## 做了什么

一次性补齐面向用户的能力（分支 `feat/webui-overhaul`，41 提交，合并入 main）：

- **设置页（新增）**：BTN 凭证/开关、GeoIP/IP 库、网络代理三块,`GET/PUT /api/config/app`,保存即**热生效**（BTN 热启停、代理热加载,无需重启）。
- **规则配置图形化**：全部 ~10 个模块改为**分组折叠式**（核心检测/可选规则），开关+表单+中文解释,字段键与引擎实际解析对齐;保留「高级 YAML」回退。后端 profile 端点支持 JSON 读写（前端无需 YAML 库）。
- **下载器并入仪表盘**：模态增删改 + 测试连接,移除独立下载器页。
- **头部**：统一品牌 `PeerBanHelper-Rust` + 版本号 + GitHub 按钮 + SVG 图标（修复退出图标缺字）+ 有新版徽标。
- **检查更新 / 一键自更新**：`/api/update/check`（走代理）+「关于」卡显式检查;`/api/update/apply` 下载本平台 `pbh-rust-<target>.gz`→解压→原子替换→自动重启（unix exec / 其它 spawn+exit,serve 加端口绑定重试）。
- **GeoIP 自动下载**（对齐上游 IPDB）：三镜像回退（github/ghostchu/paulzzh）下载 `GeoLite2-City/ASN + GeoCN`,45 天过期,**后台重试直到三库齐全**,复用有效本地缓存,下载完热加载;补 GeoCN 解析（net_type/省市）。
- **全局代理**：新 `pbh-net` crate,BTN/IP 订阅/GeoIP/检查更新统一走代理感知客户端;代理不通/未配置→直连;**下载器不走代理**。
- **本机网络地址**：设置页展示局域网 IPv4 + 公网 IPv4/IPv6（直连出口 IP,域内服务优先 ip.3322/ipw.cn/ipip/taobao,从正文提取 IP）,一键复制 IP / `/24` 网段,便于配 qB 白名单。
- **登录退避修复**：鉴权错误（401/403/密码错）**立即暂停**自动重试,只告警一次,改下载器配置后恢复 → 从源头避免硬刷 `/auth/login` 触发 qB 封 IP;网络错误维持每轮重试。
- **Release 改名**：包/可执行 `pbh` → `pbh-rust`;新增自更新资产 `pbh-rust-<target>.gz`。

## 为什么

兑现用户提出的一批需求:WebUI 缺 BTN 设置、退出图标缺失、规则页太乱、要检查/自更新、GeoIP 自动化、联网走代理、包名 pbh-rust;以及实测暴露的登录硬刷封 IP、公网 IP 取不到等问题。

## 影响的文件 / 范围

- 新 crate:`pbh-net`(代理客户端)。
- `pbh-config`:`network.proxy`。
- `pbh-geoip`:`GeoIpHandle`(arc-swap 热替换)、`download`(镜像回退/45天/原子写)、GeoCN 解析。
- `pbh-downloader`:`DownloaderError::Auth`(401/403 分类)。
- `pbh-engine`:`build_modules` 透传 proxy、登录暂停集、ip_rule_list 走 pbh-net。
- `pbh-btn`:`BtnManager` 热启停、`spawn` 返回 AbortHandle、proxy。
- `pbh-web`:设置/规则/下载器/检查更新/自更新/本机IP 端点 + 单页重写;`selfupdate` 模块。
- `pbh-server`:GeoIpHandle/BtnManager 装配 + GeoIP 后台下载循环。
- `.github/workflows/release.yml`、`build.sh`、`README.md`:pbh-rust + 自更新资产。
- 版本 0.1.0 → **0.1.1**。

## 注意事项 / 后续

- **自更新从本版起可用**:`pbh-rust-<target>.gz` 资产是本版加入发布流程的,故需先发 `v0.1.1`,之后版本间才能在 WebUI 一键自更新;且程序文件需对运行用户可写。
- **沙箱限制**:开发环境网络封 GitHub,故"GitHub 找资产/真实自我重启""真实公网 IP 拉取"未在沙箱端到端验证;核心环节(下载/解压/校验/原子替换/失败安全)已用本地服务真实测试通过。
- 全工作区测试 21 套件全绿、clippy 零告警;GeoIP 自动下载、各读写端点、登录退避均经本机实测。
- 设计与计划见 `docs/superpowers/specs|plans/2026-06-21-webui-overhaul*`。
