# M7 极简 Web API + 内置单页（浏览器可测成品）

## 做了什么
- **pbh-web**（新）：`axum` HTTP 服务。
  - `ApiResp<T>{ok,data,error}` 信封（`IntoResponse`）+ `Page<T>`。
  - Bearer token 鉴权中间件（公开 `/` 与 `/api/auth/login`，其余 `/api/*` 需匹配 config token，否则 401）。
  - 端点：`POST /api/auth/login`、`GET /api/status`、`GET/PUT /api/downloaders`、`DELETE /api/downloaders/{id}`、
    `POST /api/downloaders/test`、`GET/PUT /api/bans`、`DELETE /api/bans/{ip}`、`GET /api/bans/history`、`GET /api/logs`。
  - 内置 vanilla 单页（`assets/index.html`，`include_str!` 内嵌）：登录、下载器列表/增删改/测试连接、当前封禁+手动封/解封、封禁历史、实时日志（3s 轮询）。
- **pbh-engine**：`BanManager` 增 `manual_ban`/`manual_unban`/`global_ban_duration`。
- **pbh-server**：构造 `WebState` 并 `tokio::spawn` 启动 Web 服务（与 ban wave 并行）;启动日志打印界面 URL。

## 为什么
- 让用户能用浏览器登录、添加自己的 qB、看自动封禁——即「可运行的成品供测试」的完整形态，不必手改 YAML。

## 影响的文件 / 范围
- 新增 crates/pbh-web/{Cargo,src/{lib,envelope,routes},assets/index.html}。
- 改 pbh-engine/ban_manager.rs、pbh-server/{Cargo,main}、README、roadmap、test-status。
